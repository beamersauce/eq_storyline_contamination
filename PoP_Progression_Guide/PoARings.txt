<c "#fc7703">  < > < > Boss Info < > < > < > </c><br>
Name: Avatar of Dust<br>
Level: 75<br>
HP: 650,000<br>
Max Hit: 2000+<br>

<c "#0080ff">  < > < > Event Info < > < > < > </c><br>
AE rampage, flurries, partially slowable.<br>
Buff magic resist.<br>
Complete the spider event to make Sigismond Windwalker become active, which then spawns the Avatar of Dust (stays up for 45 minutes).<br>
15x An Erratic Arachnid (baby spiders with spin stun) instantly respawn when killed, so mezz, offtank, or kite.<br>
Kill the other (medium sized) spiders.  They split into smaller spiders each time.  The last one is A Vorladien Archwalker, which makes Sigismond targetable.<br>
Sigismond has 600,000 HP, max hit for around 2000, has a PBAE silence, and a single target 3500 DD.<br>

<c "#0080ff">  < > < > Spells Info < > < > < > </c><br>
Chain Lightning (Magic): Range 30 Targeted AE 500 damage, 500 mana drain.<br>
Wave of White Noise (Unresistable): Range 100 PBAE silence.<br>
Armor Break (Unresistable): Range 200 PBAE -270 AC.<br>

Name: Avatar of Mist<br>
Level: 75<br>
HP: 800,000<br>
Max Hit: 2400+<br>

<c "#0080ff">  < > < > Event Info < > < > < > </c><br>

<c "#0080ff">  < > < > Spells Info < > < > < > </c><br>

Stone Gale (Unresistable): Range 20 Targeted AE 1500 damage, 2 second stun.<br>

Level: 75<br>
HP: 800,000<br>

<c "#0080ff">  < > < > Event Info < > < > < > </c><br>
AE rampage, flurry, partially slowable.<br>

Hail Storm (Magic): Range 80 PBAE 400 damage, 1 second stun.<br>

Name: Avatar of Wind<br>
Level: 75<br>
HP: 750,000<br>
